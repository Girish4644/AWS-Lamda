# AWS-Lamda

Serverless Python function built with **AWS Lambda**.

## Overview

This project contains an AWS Lambda function that processes incoming events and returns a structured response.  
It is intended as a clean starter for Lambda-based APIs, automations, and event-driven workflows.

## Tech Stack

- Python 3.x
- AWS Lambda
- AWS CLI (for deployment/testing)
